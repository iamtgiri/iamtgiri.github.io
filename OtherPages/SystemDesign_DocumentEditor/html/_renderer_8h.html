class  Renderer
 The Renderer class is an abstract base class that defines the interface for rendering different types of document elements. It includes pure virtual methods for visiting various document element types, such as TextElement, ImageElement, NewLineElement, and TabElement. Additionally, it provides pure virtual methods to retrieve the final rendered result and reset the internal state for a new rendering session. Concrete implementations of the Renderer interface will provide specific logic for rendering each type of document element. More...

Detailed Description

Declaration of the Renderer interface, which defines methods for rendering different types of document elements. This interface is designed to be implemented by concrete renderer classes, such as PlainTextRenderer, to provide specific rendering logic for various document elements like text, images, new lines, and tabs. The Renderer interface also includes methods to retrieve the final rendered result and reset the internal state for a new rendering session.
