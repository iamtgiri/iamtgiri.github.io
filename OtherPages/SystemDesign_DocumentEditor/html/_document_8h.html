Detailed Description

Declaration of the Document class, which represents a collection of document elements. This class provides methods to manage and access the document elements. The Document class uses a vector of unique pointers to store the elements, ensuring proper memory management. The class includes a destructor to clean up resources and a method to add new elements to the document. The getElements method allows access to the collection of document elements for rendering or other operations.
