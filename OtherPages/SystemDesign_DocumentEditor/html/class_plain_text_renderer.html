Detailed Description

The PlainTextRenderer class is a concrete implementation of the Renderer interface, designed to render document elements as plain text. It maintains an internal buffer to accumulate the rendered output and provides methods to visit different types of document elements, such as text, images, new lines, and tabs. The class also includes methods to retrieve the final rendered result and reset the internal state for a new rendering session.
