Detailed Description

Declaration of the DocumentElement class, which represents an element in a document. This class serves as a base class for different types of document elements, such as text and images. The DocumentElement class defines a pure virtual method accept, which allows a Renderer to process the element. This design follows the Visitor pattern, enabling separation of rendering logic from the document structure.
