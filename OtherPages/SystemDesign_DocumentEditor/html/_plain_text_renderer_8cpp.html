Detailed Description

Implementation of the PlainTextRenderer class, which renders document elements as plain text. This class implements the Renderer interface and provides specific logic for rendering different types of document elements, such as text, images, new lines, and tabs. The rendered output is accumulated in an internal buffer, which can be retrieved using the result method. The reset method allows clearing the buffer for a new rendering session.
