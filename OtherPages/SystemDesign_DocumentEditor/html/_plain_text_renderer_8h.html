Detailed Description

Declaration of the PlainTextRenderer class, which implements the Renderer interface to render document elements as plain text. This class provides methods to visit different types of document elements, such as text, images, new lines, and tabs, and accumulates the rendered output in an internal buffer. The PlainTextRenderer class also includes methods to retrieve the final rendered result and reset the internal state for a new rendering session.
